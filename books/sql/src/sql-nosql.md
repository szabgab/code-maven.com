# NoSQL

[NoSQL](https://en.wikipedia.org/wiki/NoSQL) is a popular name for a lot of very different databases. The common denominator is that their primary model is not relational and their primary interface is not SQL.

A few of the popular NoSQL systems:

* [MongoDB](https://www.mongodb.com/)
* [Redis](https://redis.io/)
* [Apache Cassandra](http://cassandra.apache.org/)

Wikipedia has a large [list of NoSQL software and tools](https://en.wikipedia.org/wiki/List_of_NoSQL_software_and_tools).
