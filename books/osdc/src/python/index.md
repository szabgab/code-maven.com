# Python

Contribution to Python-based projects.

See the [OSDC Python](https://osdc.code-maven.com/python)

There are many things one can contribute, here I'll try to describe several options.

1. Add meta data: [Link to VCS](./link-to-vcs.md)
1. Add Continuous Integration (CI)
    * GitHub Actions
    * GitLab Pipeline
1. Add dependabot
1. Add tests (Generate test coverage report, find holes and create more tests)
1. Code Formatting using black?
1. Linting using flake8 or some other tool.
1. Type annotation and type checking using mypy
