# Harnesses

* [GitHub co-pilot](https://github.com/features/copilot) (Microsoft)
    * In VS Code
    * CLI
    * GitHub UI to chat
    * GitHub Issues - assign to copilot
    * Available Models (from other vendors)
      * GPT-5.4
      * GPT-5.3-Codex
      * GPT-5.2
      * GPT-5.4 mini
      * GPT-5 mini
      * GPT-4.1
      * Claude Sonnet 4.6
      * Claude Haiku 4.5

* [Cursor](https://cursor.com/)
    * A modified version of VS Code. An independent startup.

* [Codex](https://openai.com/codex/) (OpenAI)
    * In VS Code
    * CLI
* [Gemini Code Assist](https://codeassist.google/products/business) (Google)
    * In VS Code
    * Gemini CLI is deprecated
    * Antigravity CLI
* [Claude Code](https://claude.com/product/claude-code) ([Anthropic](https://www.anthropic.com/))
    * CLI - It is a command line tool.
    * It can work as an extension in major IDEs.


* [Windsurf (Codeium)](https://windsurf.com/)
    * Stand alone editor
    * Plugin to JetBrains, VS Code etc.

* [Tabnine](https://www.tabnine.com/)
