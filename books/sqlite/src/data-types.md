# Data types

* [Data types](https://sqlite.org/datatype3.html)
* [Flexible typing](https://sqlite.org/flextypegood.html) - SQLite does not really care about the types you defined for the columns. You can even create columns without types.
* [STRICT tables](https://sqlite.org/stricttables.html) - In special cases you can ask SQLite to enforce strict type-checking by setting the table to be `STRICT`.

## Basic types

* NULL
* INTEGER
* REAL
* TEXT
* BLOB
